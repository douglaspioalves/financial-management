## Interface esperada de InstallmentService

Os testes assumem a seguinte assinatura do serviço:

```java
// Método principal testado
List<Installment> generateInstallments(Transaction transaction, Card card);
```

O serviço deve:
1. Calcular `firstReferenceMonth` baseado em `purchaseDate.day` vs `card.closingDay`
2. Gerar N `Installment` com `referenceMonth = firstReferenceMonth + (i-1) meses`
3. Usar FLOOR(total/N, 2 decimais) para as N-1 primeiras parcelas
4. Última parcela = total - soma(primeiras N-1)
5. Chamar `installmentRepository.saveAll(List<Installment>)` antes de retornar
