## Problema encontrado durante a sessão

após cada operação de escrita/edição. Além disso, o git muda de branch sozinho para
`feature/s03-frontend`. Resultado: todos os commits do backend foram para `feature/s03-frontend`
em vez de `feature/s03-backend` (que tem apenas 1 commit com versão antiga/incorreta).

O código está correto e testado em `feature/s03-frontend`.
Branch `feature/s03-backend` tem apenas o commit inicial (8e1fa3b) com versão incorreta
(existsByOwnerPersonId) que causava falha no contexto Spring.
